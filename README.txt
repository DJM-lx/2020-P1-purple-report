This file contains all the necessary resources to reproduce our project for P1-2020

///////////////////////////////////////////////////////////////////////////////

COPYRIGHT.txt - A copyright provision detailing the allowed use of our project resources

2020-P1-purple-team-resources.pdf - a compiled list of resources that guided us in the brainstorming phase

2020-P1-purple-brainstorming.pdf - A PDF of our initial brainstorming presentation given at the beginning of the project cycle

2020-P1-purple-final.pdf - The final report detailing and discussing the results of the project deliverable

src - A folder containing our source code and the details for its use
	-scorpionMasterControl.py - our python script that sets up sensor and waypoint data
	-scorpionPlans.py - our python script for contains plans for autonomous movement
	-scorptionReIX.py - our python script for translating simulated motion to robotic movement
	-scorpionSimIS.py - our python script for simulated motion

howto- A folder containing our how-to documentation to recreate the robot mechanically
	-2020-P1-Purple-howto.pdf the how-to guide to recreate the robot step-by-step
	-3D_Print.zip - the 3d printing files for 3d printed parts we used
	-CadFiles.zip - the cad files for 3d printed parts 
	-Images.zip - the cad images of our robot and robot parts
	-Laser_cutting.zip - files for laser cutting the base

brstm- Source files we used for our brainstorming phase
	- 2019 Red Team's Cad files for their mechanum wheels
